# Form to Google Sheets + Email Confirmation

## Problem
Businesses manually saving contact form submissions and sending confirmation emails — slow, inconsistent, and easy to miss.

## Solution
This n8n workflow automatically saves every form submission to Google Sheets and instantly sends a confirmation email to the submitter.

## Tools
| Tool | Role |
|------|------|
| n8n Form Trigger | Captures form submissions |
| Google Sheets | Logs submissions as rows |
| Gmail | Sends confirmation email |

## Outcome
✅ Zero manual data entry. Confirmation email sent in seconds. Never miss a lead.

## Setup
1. Import `workflow.json` into n8n
2. Connect Google Sheets and Gmail credentials
3. Update the email template with your name
4. Publish the workflow
